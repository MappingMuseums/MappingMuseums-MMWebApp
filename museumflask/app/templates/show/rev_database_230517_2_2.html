Visualisation Patterns Emerging

 

(A) Number of museums in existence per (attributes A1, A2, … An, time interval)

 

- allows drilling down into one or more attributes, for more detail

- allows drilling down into the time interval, for more detail

- allows drilling down into combinations of the above

- allows selecting one or more values of an attribute Ai, and plotting a line showing the Number for each value (x axis) against time (y axis)

 

(A-qualif) Number of museums in existence satisfying filters F per (attributes A1, A2, … An, time interval)

 

(B) Number of museum events of type E per (attributes A1, A2, … An, time interval)

 

(B-qualif) Number of museum events of type E satisfying filters F per (attributes A1, A2, … An, time interval)

 

 

Possible Visualisations

 

If we are plotting just one attribute against time, e.g. Number of museums in existence per (Region, Time interval), then a 2-dimensional visualisation can be used.

Alternatively, Region can be captured as a spatial attribute (e.g. a map) and time can be captured through a slider, or through a sequence of maps at different time intervals.

 

If we are plotting two attributes against time, e.g. Number of museums in existence per (Subject, Region, Time interval), then a 3-dimensional visualisation can be used.

Alternatively, time can be captured through a slider under a 2D visualisation, or through a sequence of 2D visualisations.

 

If we are plotting three attributes against time, e.g. Number of museums in existence per (Subject, Region, Visitor Numbers, Time interval), then a hyperdimensional visualisation can be used e.g. a “Star Plot”

        Alternatively, time can be captured through a slider under a 3D visualisation, or through         a sequence of 3D visualisations

 

If we are plotting n > 3 attributes against time, e.g. Number of museums in existence per (Governance, Subject, Region, Visitor Numbers, Time interval), then a hyperdimensional visualisation can be used e.g. a “Star Plot”

Alternatively, time can be captured through a slider under a star plot (with n-1 dimensions) or through a sequence of star plots with n-1 dimensions
